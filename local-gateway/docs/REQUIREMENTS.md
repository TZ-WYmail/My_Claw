# LocalCommandCenter 用户需求文档

> 基于用户视角的反复分析，识别出的缺失功能与优化项

## 一、需求分析思路

以真实用户身份使用当前系统，逐步发现以下痛点：

1. **打开页面不知道系统状态** — 当前只有任务管理，没有全局概览
2. **下载完文件找不到** — 没有下载历史记录，只有文件搜索
3. **下载的文件只能看路径** — 没有文件预览或管理功能
4. **任务多了找不到** — 没有搜索/筛选功能，周计划之外的看不到
5. **操作没有反馈** — 看不到系统运行日志，出问题只能猜
6. **效率低** — 没有键盘快捷键，没有全局搜索

---

## 二、需求清单（按优先级排序）

### P0 — 核心缺失功能

| # | 需求 | 问题描述 | 实现方案 |
|---|------|----------|----------|
| 1 | **仪表盘 Dashboard** | 打开页面无法一眼看到系统整体状态 | 新增 Dashboard 首页标签，展示：任务统计、下载统计、磁盘占用、最近活动 |
| 2 | **下载历史记录** | 下载完成后无持久化记录，关闭页面就忘了 | 后端新增 `download_history` 数据表 + `/api/download/history` 端点 + 前端历史列表页 |
| 3 | **全部任务视图** | 目前只有「本周计划」，看不到历史和未来任务 | 后端新增 `get_all_tasks` action + 前端任务列表带分页和筛选 |
| 4 | **操作日志** | 系统做了什么完全不知道，出问题无法排查 | 后端新增 `logs` 表记录所有操作 + `/api/logs` 端点 + 前端日志面板 |

### P1 — 体验优化

| # | 需求 | 问题描述 | 实现方案 |
|---|------|----------|----------|
| 5 | **任务搜索与筛选** | 任务多了无法快速定位 | 前端搜索框 + 按状态/周期筛选 |
| 6 | **文件操作按钮** | 搜索到文件只能看路径，无法操作 | 搜索结果添加「打开目录」「删除」按钮 |
| 7 | **下载进度实时显示** | 大文件下载看不到进度 | WebSocket 或 SSE 推送 + 前端进度条 |
| 8 | **暗色/亮色主题切换** | 用户可能不喜欢暗色 | CSS 变量 + 主题切换按钮 |

### P2 — 高级功能

| # | 需求 | 问题描述 | 实现方案 |
|---|------|----------|----------|
| 9 | **数据导出** | 任务列表和下载记录无法导出 | 导出为 JSON/CSV |
| 10 | **快捷键** | 操作效率低 | Ctrl+N 新建任务 / Ctrl+F 搜索 / Tab 切换标签 |
| 11 | **移动端适配优化** | 当前响应式但不够精细 | 移动端专用布局 |
| 12 | **系统设置面板** | 无法在 UI 中调整配置 | 端口、超时、目录等可视化配置 |

## 三、实现计划

### 阶段 1：核心功能（本次实现）

1. ✅ 仪表盘 Dashboard — 统计卡片 + 最近活动 + 系统状态
2. ✅ 下载历史记录 — 后端持久化 + 前端历史列表
3. ✅ 全部任务视图 — 分页 + 搜索 + 状态筛选
4. ✅ 操作日志 — 自动记录所有操作 + 可查看
5. ✅ 主题切换 — 暗色/亮色一键切换
6. ✅ 键盘快捷键 — Ctrl+N/F/K 等

### 阶段 2：体验增强（后续迭代）

7. 下载进度实时推送 (SSE)

10. 系统设置面板

## 四、技术方案

### 后端改动

1. **数据库新增 2 张表**：
   - `download_history` — 记录每次下载的 URL、文件名、分类、大小、时间、安全扫描结果
   - `operation_logs` — 记录所有 API 调用的操作类型、参数、结果、时间

2. **新增 API 端点**：
   - `GET /api/dashboard` — 仪表盘统计数据
   - `GET /api/download/history` — 下载历史列表
   - `POST /api/task` 新增 `get_all_tasks` action — 全部任务带筛选
   - `GET /api/logs` — 操作日志列表
   - `GET /api/export/{type}` — 数据导出

3. **中间件**：
   - `LoggingMiddleware` — 自动记录所有 API 请求到日志表

### 前端改动

1. 新增 Dashboard 标签页
2. 改造下载面板 — 添加下载历史 Tab
3. 改造任务面板 — 添加搜索/筛选 + 全部任务 Tab
4. 新增日志标签页
5. Header 添加主题切换 + 全局搜索
6. 注册键盘快捷键
